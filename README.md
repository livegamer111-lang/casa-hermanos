# Casa Hermanos

Ready-to-upload Vercel React website.

## Files must be at GitHub repository root

Your repository should show:

- index.html
- package.json
- tailwind.config.js
- postcss.config.js
- src/App.jsx
- src/main.jsx
- src/index.css

## Vercel settings

Framework Preset: Vite  
Build Command: npm run build  
Output Directory: dist
